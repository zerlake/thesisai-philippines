## Score Calculation Hints

Scores are based on:

**Novelty**: 
- How recent is related research?
- How many studies address this exact gap?
- How specific is your gap framing?

**Feasibility**:
- Can you access the population?
- Do you have time? (1-2 years)
- Do you have resources? (budget, equipment, data)
- Are there ethical barriers?

**Significance**:
- Does it solve a real problem?
- Will it influence practice or policy?
- Does it benefit people?
- Is there broader application?

## Pro Tips

1. **Don't obsess over perfect scores** - Balance matters
2. **Feasibility is NON-NEGOTIABLE** - Can't finish = Can't graduate
3. **Significance justifies effort** - Why work on this if it doesn't matter?
4. **Use "Validate" tab next** - Check if your gap statement is clear
5. **Talk to advisor** - They know feasibility constraints
6. **Compare all gaps** - See which scores make the best combination
