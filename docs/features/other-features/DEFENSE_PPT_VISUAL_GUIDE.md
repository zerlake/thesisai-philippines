## Troubleshooting

### Sample Buttons Not Showing?
- Refresh the page
- Clear browser cache
- Make sure you're on the Setup tab (Step 1)

### Sample Loads but No Slides?
- Wait a moment for slides to render
- Check browser console for errors
- Try a different sample

### Keyboard Shortcuts Not Working?
- Make sure presentation has focus (click it first)
- Some keyboard shortcuts may be blocked by browser
- Try using the button controls instead

### Sample Data Not Saved?
- Samples are for viewing/editing
- Save your work to localStorage via "Copy as Template"
- Or use the Save button if available
