## Contributing to Wiki

To add or update wiki pages:

1. **Create/edit .md file** in `/docs/wiki/`
2. **Follow naming convention** - PascalCase with hyphens (e.g., Feature-Name.md)
3. **Link from INDEX.md** - Add to appropriate section
4. **Include diagrams** - Use ASCII art or descriptions
5. **Add examples** - Real code when possible
6. **Update Home.md** - If new major topic
