üí° CODE SPLITTING EXAMPLE
================================================================================

Before (All in bundle):
  import Dashboard from '@/components/document-analyzer';
  
After (Lazy loaded):
  import dynamic from 'next/dynamic';
  const Dashboard = dynamic(
    () => import('@/components/document-analyzer'),
    { loading: () => <Skeleton /> }
  );

Impact: Only loads when needed, reduces initial bundle
