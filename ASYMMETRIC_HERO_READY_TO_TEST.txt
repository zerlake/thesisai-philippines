================================================================================
TROUBLESHOOTING
================================================================================

Issue: "HeroSection is not defined" error
Solution:
  1. Clear Next.js cache: rm -rf .next .turbo
  2. Restart development server: pnpm dev
  3. Hard refresh browser: Ctrl+Shift+R (Chrome) or Cmd+Shift+R (Mac)

Issue: Images not loading in carousel
Solution:
  1. Check /public directory for image files
  2. Verify filenames match:
     - hero-carousel-research.webp
     - hero-carousel-ai.webp
     - hero-carousel-thesis.webp
  3. System will use fallback Unsplash URLs if local images missing
  4. Check browser console for 404 errors

Issue: Carousel not rotating after 5 seconds
Solution:
  1. Check browser console for JavaScript errors
  2. Verify CAROUSEL_INTERVAL = 5000ms in hero-carousel.tsx
  3. Open DevTools → Network tab → verify images loading
  4. Check that mounted state is true in React DevTools

Issue: Animations are laggy/choppy
Solution:
  1. Check Lighthouse performance score
  2. Open DevTools → Rendering → disable "Paint flashing"
  3. Check prefers-reduced-motion setting
  4. Verify GPU acceleration is enabled
  5. Look for layout shifts (CLS)

Issue: Responsive layout broken on mobile
Solution:
  1. Test with DevTools device emulation
  2. Check Tailwind responsive classes (md:, lg:)
  3. Verify viewport meta tag in layout.tsx
  4. Clear Tailwind cache: rm -rf .next
