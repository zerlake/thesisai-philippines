# Author Collaboration Network - Quick Reference

## What Is It?
Visual map showing how researchers collaborate. Each author is a node, co-authorship is a connection.

## Key Visuals
| Visual | Meaning |
|--------|---------|
| **Large node** | Author published many papers in search results |
| **Small node** | Author has 1-2 papers in search results |
| **Green node** | Author collaborates with many others |
| **Blue node** | Author has few collaborators |
| **Line between nodes** | Co-authorship relationship |

## How It Helps You Find Papers

**Problem:** 100 papers by 50 different authors - hard to organize
**Solution:** Author Network shows you:
1. Who the main researchers are (large nodes)
2. Who works together (connected clusters)
3. Which groups focus on similar topics
4. How the research community is structured

## Basic Usage

| Action | Result |
|--------|--------|
| **Hover over node** | See author name, paper count, connections |
| **Click node** | Select author (notification shows paper count) |
| **Drag canvas** | Pan to explore different areas |
| **Use Zoom buttons** | Get closer or see whole network |
| **Click Play/Pause** | Pause animation to examine static view |

## 3-Step Research Workflow

### Step 1: Search & Analyze
- Search for your topic (10+ papers needed)
- Open "Author Network" tab
- Study the network pattern

### Step 2: Identify Key Researchers
- Look for large green nodes (many papers, many collaborations)
- These are influential researchers in your field

### Step 3: Systematic Collection
- Click each major author
- Add their papers to collection
- You've now systematically covered your field

## Common Issues & Fixes

| Problem | Cause | Solution |
|---------|-------|----------|
| "No author connections" | No papers or no author data | Search for papers first |
| Very sparse network | Too few papers | Search with broader terms |
| Can't see details | Zoomed too far out | Use Zoom In or Reset View |
| Network too cluttered | Too many authors | Zoom in on specific cluster |
| Can't identify patterns | Animation moving too fast | Click Pause button |
